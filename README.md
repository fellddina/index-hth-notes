# index-hth-notes

## 项目简介

本仓库用于归档和发布一系列独立的 HTML 页面。这些页面通常用于记录、整理或展示特定主题下的信息，以静态网页形式存在，方便直接浏览和分享。

仓库本身不绑定任何特定域名或网站，仅作为页面文件的存储与索引中心。

## 目录说明

```
index-hth-notes/
├── README.md          # 本文件
├── pages/             # 存放所有独立 HTML 页面
│   ├── example.html
│   └── ...
└── assets/            # 页面使用的静态资源（可选）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：所有 HTML 页面均放置在此目录下，每个页面为一个独立文件。
- `assets/`：若页面需要引用外部样式、脚本或图片，建议统一存放于此。

## 页面归档说明

每个 HTML 页面均为独立完整的文档，可直接在浏览器中打开使用。页面内容可能涵盖笔记、工具、演示、参考信息等，不局限于单一主题。

- 页面命名建议使用英文小写、连字符分隔，便于识别与链接。
- 页面内部可包含简单的样式与脚本，但应避免对外部服务的强依赖。
- 若页面需要引用仓库内资源，请使用相对路径。
